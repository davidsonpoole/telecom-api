# telecom-api

The API for an Alexa app designed to automate customer service calls. Returns data from the AWS database (now inactive) like billing info, usage, and viewing preferences.

**Endpoints:**
- /viewBill
- /viewUsage
- /payBill
- /viewUserInfo
- /register
- /deleteUser
- /viewAllUsers
- /upgrade
- /downgrade
- /parentalControls
- /setViewingPrefs
- /setInterests
- /viewInterests
- /viewPrefs
- /viewOutage
